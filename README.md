## 搬瓦工续费的完整时间线

理解续费，先理解时间线。账单是什么时候出来的，又有多久的付款窗口，这两个数字决定了你有多少时间反应。

**到期当天**：账单自动生成。搬瓦工计费系统用的是北美时区，所以国内用户在到期日的第二天凌晨到早上才会看到账单出现，有时候会觉得"过了一天了还没出账单"，其实就是时差。

**账单生成后**：系统发邮件，标题通常是 Invoice Created，邮件里有账单详情和付款链接。这封邮件建议设置一下 QQ 邮箱的白名单或者用 Gmail 接收，防止被归入垃圾箱。

**7 天宽限期**：账单出来之后，VPS 还会继续正常运行 7 天，这段时间是你的付款窗口。不夸张，7 天完全够用了。

**7 天后仍未付**：VPS 被暂停（Suspend）。数据还在，账单通常也还有效，付款之后可以恢复。

**再拖下去**：续费账单被取消，VPS 连同数据彻底删除，无法恢复。这个时间点官方没有精确说明，所以不要去赌。

总结成一句话：**到期当天出账单，7 天内付款，VPS 全程不中断。**

逻辑很简单：账户里的余额（Credit）会优先用于支付账单。只要到期前往账户里充入足够的金额，账单一生成，系统会自动从余额扣款，续费完成——全程不需要人工干预。

## 续费价格会涨吗？老用户必看的价格锁定机制

这是很多人最担心的一点，怕哪天搬瓦工涨价，续费的时候多扣钱。

实际情况是：续费价格按**首购价格锁定**，和后续官网定价无关。你当初买的是多少，后续续费就是多少。即便官方把同款套餐的价格调高了，你的账单也不受影响。

首购时用了优惠码的话，续费时优惠码也会自动继续生效，折扣不会丢。但有一点要注意——续费时不能更换新的优惠码，只能延续首购时绑定的那个。

**有一个操作会打破价格锁定，需要特别当心：修改续费周期。**

比如从月付改成年付，或者反过来，系统会按当前官方定价重新计算。如果你当初买的是很早的老方案，价格远低于现在，随意修改周期可能导致续费金额大幅上升。

我自己因为懒，一直用年付没动过，续费价格这几年都没变过。

## 续费失败的几种常见情况及处理方法

续费这件事本来很简单，但偶尔还是会碰到一些让人摸不着头脑的情况。整理了几个高频问题：

**情况一：VPS 快到期了，后台没有续费按钮**

不是问题。账单到期当天才生成，到期之前就是没有续费入口，这是正常设计。等到期那天看后台或者查邮件就好。

**情况二：到期了但账单迟迟没出来**

搬瓦工用的是北美时区，可能比北京时间晚 15-16 小时左右。觉得"明明到期了"但账单没出，多等几个小时通常就有了。

**情况三：没收到续费提醒邮件**

先查垃圾邮件箱。很多国内邮箱会把来自境外的系统邮件直接拦截。另一个方法是直接去后台看：Billing → My Invoices，所有未支付账单都在这里，不依赖邮件。

**情况四：VPS 已经被暂停了**

先去后台确认续费账单是否还存在。只要 Unpaid 账单还在，支付之后 VPS 就能立刻恢复，数据不会丢。

**情况五：后台显示 Cancelled，续费入口消失了**

通常是 OpenVZ 架构的老套餐，这类机器搬瓦工已经全面停用，到期后无法续费，只能重新购买 KVM 套餐。建议提前做好数据迁移。

## FAQ：关于搬瓦工续费的高频问题

**Q1：搬瓦工会不会自动扣我的支付宝或银行卡？**

不会。搬瓦工官方明确说明不存储用户支付信息，技术上也无法自动扣款。唯一的自动扣款方式是你主动往账户余额里充了钱。

**Q2：我可以手动在续费前一个月去续费吗？提交工单申请可以吗？**

两种方式都行不通。账单出来之前既没有付款入口，客服也不受理提前续费申请。变通方式是提前充值余额，等账单自动触发扣款。

**Q3：续费的时候能换用新的优惠码吗？**

不能。续费只延续首购时绑定的优惠码，无法在续费时更换或新加优惠码。

**Q4：有多台 VPS，账单会合并吗？**

不会。每台 VPS 独立生成账单，需要分别支付。

**Q5：我的续费价格比官网现价低很多，正常吗？**

正常，这是价格锁定机制。只要没改变续费周期，首购价就是永久续费价，不受官网涨价影响。

**Q6：如果我打算不用了，到期不续费会怎样？**

到期后 7 天内 VPS 仍可正常访问，7 天后暂停，再过一段时间账单失效后 VPS 和数据彻底删除。不需要主动操作什么，不付款就行。注意不建议关掉"Renewal: ON"开关——保持 ON 不会自动扣款，关掉 OFF 之后如果反悔想续费反而操作更麻烦。
